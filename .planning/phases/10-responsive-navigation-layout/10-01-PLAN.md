<task type="auto">
  <name>Task 1: Restructure base.html for hamburger + drawer navigation</name>
  <files>app/templates/base.html</files>
  <action>
  Modify `app/templates/base.html` to restructure the navigation. The new HTML structure inside `<nav class="nav">` should be:

  **Top bar (always visible):**
  - Brand link "BeanBay" (left) — keep existing
  - Hamburger button (right) — new, `<button class="nav-hamburger" id="nav-toggle" aria-label="Toggle navigation" aria-expanded="false">` containing a 3-line hamburger icon using a `<span class="hamburger-icon"></span>` (CSS-only, no SVG needed)

  **Drawer (off-screen by default):**
  - `<div class="nav-overlay" id="nav-overlay"></div>` — semi-transparent backdrop
  - `<div class="nav-drawer" id="nav-drawer">` containing:
    - All 4 nav links (Brew, History, Insights, Analytics) as `<a class="nav-drawer-link">` elements, stacked vertically
    - A divider `<hr class="nav-drawer-divider">`
    - The active bean indicator section (the existing `nav-active-bean` div with its full content including the deactivate form)

  **Key details:**
  - Remove the `.nav-links` div and `.nav-active-bean` div from the top bar — they move into the drawer
  - Remove the `.container` class from `.nav-inner` — the top bar should be full-width with its own padding (so it's not constrained to 480px)
  - The `.nav-inner` becomes just the top bar: brand + hamburger, using `display: flex; justify-content: space-between; align-items: center; padding: 0 var(--spacing-md);`
  - Add a small inline `<script>` at the end of `<body>` (before `{% block scripts %}`) for the toggle logic:
    ```javascript
    (function() {
      var toggle = document.getElementById('nav-toggle');
      var drawer = document.getElementById('nav-drawer');
      var overlay = document.getElementById('nav-overlay');
      function open() {
        drawer.classList.add('open');
        overlay.classList.add('open');
        toggle.setAttribute('aria-expanded', 'true');
        document.body.style.overflow = 'hidden';
      }
      function close() {
        drawer.classList.remove('open');
        overlay.classList.remove('open');
        toggle.setAttribute('aria-expanded', 'false');
        document.body.style.overflow = '';
      }
      toggle.addEventListener('click', function() {
        drawer.classList.contains('open') ? close() : open();
      });
      overlay.addEventListener('click', close);
    })();
    ```
  - Do NOT use any CSS framework or external JS. Vanilla only.
  </action>
